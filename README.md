# Aerowin – Aviator ✈️

A premium, modern Aviator game platform built with vanilla HTML/JS/CSS and Supabase.

## 🚀 Features
- **Modern Modular Structure**: ES Modules used throughout for maintainability.
- **Supabase Integration**: Persistent balance, authentication, and game history.
- **Premium UI**: Betika-style aesthetic with rich animations and sounds.
- **Audio Engine**: Synthesized game sounds for an immersive experience.

## 📝 Setup
1. **GitHub**: Import your repository into Vercel.
2. **Supabase**: 
   - Execute the SQL schema (see below).
   - Update your project ID and Anon Key in `src/lib/supabase.js`.

### SQL Schema
Run this in your Supabase SQL Editor:
```sql
create table public.profiles (
  id uuid references auth.users on delete cascade primary key,
  phone text unique,
  balance numeric default 1000,
  referral_code text unique,
  referred_by text,
  free_bet_given boolean default false,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

create table public.game_history (
  id bigint generated by default as identity primary key,
  multiplier numeric not null,
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);
```
